# 对数变换

变换形式如下

$$T(r) = c\lg(r+1)$$

- `c`为常数

由于**对数函数的导数**随自变量的增大而减小，对数变换将**输入窄范围的低灰度值扩展为范围宽的灰度值和宽范围的高灰度值压缩为映射为范围窄灰度值**。从视觉上，通常是图片变得更亮了

### 运行
```python
python TEST_logarithmic_transformation.py
```

# 伽马变换
